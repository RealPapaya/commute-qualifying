# Goal

Support route deviation recovery and manual start control during a timed run.

# Acceptance criteria

- When an active runner deviates from the route, the app can ask whether to create a new route.
- Accepting creates/continues a route from the current position without resetting elapsed time.
- Near the starting point, the runner can manually restart or start before automatic start detection.
- Existing timing, route recording, and relevant UI tests continue to pass.
